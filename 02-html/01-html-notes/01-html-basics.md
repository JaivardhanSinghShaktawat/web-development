# HTML Basics

## How website works ?

User makes a request to the server by using url.The server generally returns the HTML , CSS and JavaScript files as response.

**Note :** We can change the website content inside the browser via DOM through chrome developer tools.

**Note :** Ctrl + U can be used to view the source code.

## Building block of website

**HTML :** it is used to structure the content of the web page.

**CSS :** it is used to make website beautiful and responsive.

**JavaScript :** it adds interactivity and makes web page dynamic.Ex : when we click on a button than something happens.

## What is HTML ?

HTML stands for HyperText Markup Language.It is used to structure the web page.

**Syntax:**

```html
<openingTag>This is content </closingTag>
```

**Ex:**

```html
<h1>Hello World</h1>
```

## What is HTML Element ?

The combination of opening tag , content and closing tag is called as html element.

**HTML Element = Opening Tag + Content + Closing Tag**

**Opening Tag** : HTML tag name wrapped in less than and greater than sign.

```
<openingTag>
```

**Closing Tag** : HTML tag name wrapped in less than / and greater than sign.

```html
</closingTag>
```

## What is HyperText ?

- It is the text that contains links to other text.
- It is the text that contains links that connects one or more web pages together.
- HyperText and HyperLinks are same only.

## What is Markup Language ?

It is some syntax that tells the browser how the content should be organized and displayed.

## History Of HTML

Early Beginnings (1980)

The concept of hypertext was proposed by computer scientist Ted Nelson in 1960s.

In the 1980's , Tim Berners-Lee , a British computer scientist , introduced the world
wide web (WWW) and developed the first HTML.

```
HTML 1.0 (1980)
HTML 2.0 (1995)
HTML 3.2 (1997)
HTML 4.0 (1997-1999)
HTML 5.0 (2014)
```

HTML4 introduced significant improvements , including better support for forms ,
scripts and stylesheets.It was a crucial set towards modern web development.

HTML5 (2014) : HTML5 was a major milestone , with focus on multimedia , improved
semantics , and better support for web applications.It introduced new elements like
&lt;audio&gt; , &lt;video&gt; , and &lt;picture&gt;.HTML5 made web development more versatile and
capable.

**Note :** HTML and HTML5 are same only.HTML5 is the latest version or update of the
HTML.

HTML is the markup language that we use in practice and HTML5 is the latest
version of HTML.

Prior to HTML5 , we had to use the external softwares to embed videos and audio
in HTML document.
