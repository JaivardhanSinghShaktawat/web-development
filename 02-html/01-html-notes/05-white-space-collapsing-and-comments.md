# White Space Collapsing And HTML Comments

## What is White Space Collapsing ?

HTML ignores all the whitespaces and line breaks and converts them into a single space.This is called as white space collapsing.

## What are Comments ?

Comments are ignored by the browser and are not executed.They are used to take notes or make code more readable.
Commenting in HTML allows developers to leave notes about their code and functionality.
