# Fundamentals Of Web

## What is Web Development ?

The development of the websites and web applications are called as web development.

**Client :** It is the device that is requesting something from the server.

**Server :** It is the computer that sends something as a response to the client.

Generally , server sends HTML , CSS and JavaScript files to the Client as a response.
Client/Browser renders the HTML , CSS and JavaScript files into a web page.

## Frontend

It contains all the UI that we user see.

## Backend

It contains the backend logic or bussiness logic which is sensitive like authentication , authorization etc.

## Frontend Developer

Developers who have knowledge and knows how to create frontend web applications.

## Backend Developer

Developers who have knowledge and knows how to create backend web applications.

## Fullstack Developer

Developers who have knowledge about frontend and backend.

## What is MERN Stack ?

```
M : MongoDB (it is a database that is used to store data)
E : ExpressJs (it is the nodejs framework)
R : ReactJs (it is a client side javascript library)
N : NodeJs (it is a javascript runtime)
```

MERN stack is used to create fullstack web applications.
